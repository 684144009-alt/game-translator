# Game Translator

แปลเกมแบบเรียลไทม์บน iOS/iPadOS โดยใช้:
- ✅ Vision Framework (OCR on-device)
- ✅ Apple Translate Framework (แปลภาษา on-device)
- ✅ Screen Broadcast Extension (อัดหน้าจอ)
- ✅ PiP (แสดงคำแปลแบบลอยตัว)

## วิธีใช้งาน

1. กดปุ่มอัดหน้าจอใน Control Center
2. เลือก "Game Translator"
3. เริ่มเล่นเกม → คำแปลจะแสดงใน PiP

## ข้อจำกัด

- ต้องใช้ iOS 15 ขึ้นไป
- ต้องลงทะเบียนอุปกรณ์กับ Apple ID ครั้งแรก (ดูใน Wiki)
- แอปหมดอายุทุก 7 วัน (ต้องติดตั้งใหม่)
